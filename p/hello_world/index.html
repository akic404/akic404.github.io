一些常用指令

  • git init
    初始化
  • git git remote add origin https://github.com/akic404/akic404.github.io.git
    绑定仓库
  • git add .
    添加全部文件到暂存区
  • git commit -m “XXXXX”
    将暂存区内容添加到仓库中
  • git push -u origin master
    上传远程代码并合并
  • git branch
    查看分支
  • git checkout -b iss53
    增加分支
  • git checkout doc
    切换到doc
  • git push -u origin doc
    上传远程代码并合并
  • git pull origin master:brantest
    将远程主机 origin 的 master 分支拉取过来，与本地的 brantest 分支合并
  • git pull origin master
    如果远程分支是与当前分支合并，则冒号后面的部分可以省略
  • git clone –branch 分支名 –single-branch 仓库地址
    如果想只单纯的clone 指定分支，不包含其他分支信息，可以这样
  • git push origin –delete doc
    需要先切换到另一个分支，才能删除原来所处分支
